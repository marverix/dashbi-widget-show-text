# dashbi-widget-show-text

Simple Dasbhi widget that shows text.

## Usage

Widgets takes always the newest record from source. It supports data types: `String`, `Array` and `Object`.

### Example

```js
dashbiLayout.addWidget({
  name: 'show-text',
  title: 'Demo',
  source: {
    // here some source
  }
});
```
